ReactJS. Компонент

Все в ReactJS оформляется как компонент.

ReactJS. Компонент

Компонент – обернутый js объект. 
var FixedDivComponent = React.createClass({
  render: function() {
	return (<div style = {{width:"10px", height:"10px"}} />)      
  }
});

ReactJS. Компонет. Передача данных

var DinamicDivComponent = React.createClass({
    propTypes:{/*Объявляем параметры*/}
    render: function() {
        return( <div style={{ width  : this.props.width, height : this.props.height}}>
                    this.props.message
                </div> )
    }
});

ReactJS. Объявление параметров

propTypes: {
    width   : React.PropTypes.string,
    height  : React.PropTypes.string,
    message : React.PropTypes.string,
}
